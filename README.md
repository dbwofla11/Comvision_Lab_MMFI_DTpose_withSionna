# Comvision_Lab_MMFI_DTpose_withSionna

This repository contains research and implementation for WiFi CSI-based 3D Human Pose Estimation using the MMFI dataset, DT-Pose framework, and NVIDIA Sionna RT simulation environment.

## Overview

* MMFI multimodal dataset preprocessing and analysis
* DT-Pose architecture implementation and experimentation
* Wireless channel simulation using Sionna RT
* CSI-based human pose estimation research
* RF sensing and ray-tracing environment experiments

## Tech Stack

* Python
* PyTorch
* TensorFlow
* NVIDIA Sionna RT
* WiFi CSI Signal Processing

## Goal

The goal of this project is to explore robust 3D human pose estimation using wireless sensing and simulated RF environments instead of traditional camera-based approaches.
