# 실제 주거 환경의 전파 간섭 극복을 위한 시뮬레이션 기반 CSI 환경 증강 기법

## 1. 연구 배경

WiFi CSI 기반 3D 포즈 추정 모델은 특정 환경에서 수집된 데이터에 대해 높은 성능을 보이지만, 실제 주거 환경에서는 가구 배치, 장애물 이동, 반사체 변화 등의 영향으로 성능이 크게 저하된다.

특히 MMFi 및 DT-Pose와 같은 모델은 학습 환경과 테스트 환경이 달라질 경우 Cross-Environment Gap이 발생하며, 이는 실환경 적용의 주요 장애요인으로 작용한다.

본 연구는 물리 기반 전파 시뮬레이터(Sionna RT)를 활용하여 다양한 환경 변화를 생성하고, 이를 실제 CSI 데이터에 반영하는 환경 증강(Environment Augmentation) 기법을 제안한다.

---

## 2. 연구 목표

본 연구의 목표는 사람 포즈를 새롭게 생성하는 것이 아니라, 실제 CSI 데이터에 다양한 전파 환경 변화를 주입하여 환경 변화에 강건한 WiFi 기반 3D 포즈 추정 모델을 구축하는 것이다.

이를 위해 물리 기반 시뮬레이션 환경에서 장애물 이동, 반사체 추가, LOS 차단 등의 환경 변수를 생성하고, CSI 변화 패턴을 추출하여 실제 데이터셋에 적용한다.

---

## 3. 제안 방법

### 3.1 Environment Variation 생성 -> 진행중 

Sionna RT 환경에서 다음과 같은 환경 변수를 생성한다.

* 장애물 위치 변화
* 장애물 개수 변화
* 장애물 재질 변화
* LOS 차단 여부
* 반사체 추가 및 제거

환경 변화에 따른 전파 경로 변화와 채널 응답 변화를 추출한다.

---

### 3.2 CSI Perturbation 생성

각 환경 상태에 대해 다음 정보를 수집한다.

* Channel Impulse Response (CIR)
* Amplitude 변화량
* Path Count 변화량
* Delay Spread 변화량

이를 Environment Variation Metadata로 저장한다.

---

### 3.3 MMFi 데이터 증강

MMFi CSI 데이터에 시뮬레이션 기반 Perturbation을 적용한다.

CSI_aug = CSI_original + P(CSI, EV)

여기서 EV는 Environment Variation을 의미한다.

Pose Ground Truth는 그대로 유지한다.

---

### 3.4 DT-Pose 재학습

원본 MMFi 데이터와 증강 데이터를 함께 사용하여 DT-Pose를 재학습한다.

Train Set

* E01 Original
* E01 Augmented

Test Set

* E02
* E03
* E04

## 5. 기대 효과

본 연구는 실제 환경을 완벽히 재현하는 것이 아니라 다양한 환경 변화를 물리 기반 시뮬레이션으로 생성하여 학습 데이터의 환경 다양성을 증가시키는 것을 목표로 한다.

이를 통해 WiFi CSI 기반 3D 포즈 추정 모델의 Cross-Environment Generalization 성능을 향상시키고, 실제 주거 환경에서의 활용 가능성을 높일 수 있을 것으로 기대된다.
