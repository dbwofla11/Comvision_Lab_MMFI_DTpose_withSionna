## 1. 연구 목표

본 연구의 목표는

기존 WiFi CSI 기반 3D Pose Estimation 모델이 가지는

Cross-Environment 취약성을 개선하는 것이다.

특히 기존 DT-Pose가:

- 특정 환경(room)
- 고정된 multipath
- 정적 propagation pattern

에 과적합되는 문제를 해결하기 위해,

Sionna 기반의 물리 시뮬레이션을 활용하여

현실적인 전파 변화(dynamic propagation variability)를 생성하고,

이를 학습 과정에 반영했을 때 unseen environment(E04)에서의 robustness가 향상되는지를 검증한다.

# 6. 제안 방법 (Proposed)

## 핵심 아이디어

단순 random noise injection이 아닌,

```
현실적인 propagation variability
```

를 물리 기반 시뮬레이션(Sionna)으로 생성하여

학습 데이터에 반영한다.

---

# 7. Sionna 기반 Perturbation 구성

## 시뮬레이션 요소

| 요소 | 목적 |
| --- | --- |
| 추가 반사체 | multipath 변화 |
| obstacle 이동 | dynamic reflection |
| extra transmitter | WiFi interference |
| attenuation 변화 | signal fluctuation |
| TX/RX 위치 변화 | channel response 변화 |

---

## 핵심 개념

```
환경 자체를 제거하는 것이 아니라,
환경 변화에도 유지되는 motion-related representation을 학습시키는 것
```

# 8. 실험 구성

# Experiment A — Baseline

## 목적

기본 DT-Pose 성능 측정.

## 구성

```
Train:
E01~E03 clean CSI

Test:
E04 clean CSI
```

## 평가 지표

- MPJPE
- PA-MPJPE
- PCK

---

# Experiment B — Random Noise Perturbation

## 목적

단순 random corruption과의 비교.

## 구성

```
Train:
E01~E03
+ random Gaussian / phase perturbation

Test:
E04 clean CSI
```

## 의미

```
"아무 noise나 넣는다고 robustness가 증가하는 것은 아니다"
```

를 검증하기 위한 control group.

---

# Experiment C — Sionna Perturbation (Proposed)

## 목적

물리 기반 propagation variability의 효과 검증.

## 구성

```
Train:
E01~E03
+ Sionna-based perturbation

Test:
E04 clean CSI
```

# 10. 기대 결과

## 예상 결과

| 방법 | 예상 경향 |

| Baseline | E04에서 성능 저하 |

상황을 반복적으로 생성 가능.
