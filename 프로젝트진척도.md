# Phase 1 — MM-Fi Base Validation - 진행중

우선 우리가 만든 사전학습 모델이 제대로 작동하는지부터 확실히 검증하는 단계.

## 무엇을 하나요?

* MM-Fi 데이터셋 내부에서 Train/Test 분리
* 사전학습 모델 학습 및 평가 진행

## 어떻게 검증하나요?

* MPJPE(Mean Per Joint Position Error) 측정
* 예측 3D Keypoint와 Ground Truth 비교
* Skeleton 형태가 정상적으로 복원되는지 시각적으로 확인

---

# Phase 2 — Domain Gap 증명 ⭐️ (핵심 단계)

프로젝트의 핵심 문제 제기를 수행하는 단계.

## 무엇을 하나요?

MM-Fi 환경에서 학습된 모델을 실제 주거 환경 데이터에 직접 적용.

### 적용 환경

* Sionna RT 시뮬레이션 환경
* ESP32-S3 실측 CSI 데이터
* 실제 실내 Multipath 환경

## 어떻게 검증하나요?

실험실 환경과 실제 환경의 차이로 인해 성능 저하가 발생하는지 확인.

### 예상 원인

* 벽 재질 차이
* 가구 반사
* Multipath 간섭
* 실제 RF 노이즈

## 결론

> 기존 데이터셋만으로 학습된 모델은 실제 환경에서 일반화 성능이 크게 저하된다.
